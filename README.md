# SOHO Network – Cisco Packet Tracer

## Overview
This project demonstrates a Small Office/Home Office (SOHO) network designed and configured using Cisco Packet Tracer.

## Network Features
- VLAN segmentation (Users, Printers, Management)
- Inter-VLAN routing 
- Static IP addressing for infrastructure devices
- Basic network security practices

## Devices Used
- Cisco Router (2811)
- Cisco Switch (2960)
- PCs
- Network printer

## IP Addressing
- VLAN 10 – Users: 192.168.10.0/24
- VLAN 20 – Printers: 192.168.20.0/24

## How to Open
1. Download the `.pkz` file
2. Open it using Cisco Packet Tracer (v8.x or newer)

## Skills Demonstrated
- Networking fundamentals
- VLANs
- Subnetting
- Troubleshooting connectivity issues
